# حلقات تحفيظ الرياض

تطبيق React Native (Expo) بسيط لاكتشاف حلقات تحفيظ القرآن والتلاوة في الرياض والانضمام إليها — بالعربية واتجاه RTL فقط، بدون تسجيل حساب.

## التشغيل

```bash
npm install
npm start
```

ثم افتح على جهازك عبر Expo Go، أو:

```bash
npm run web
npm run android
npm run ios
```

## الشاشات

1. **الرئيسية** — قائمة الحلقات القادمة مع فلتر بالحي
2. **تفاصيل الحلقة** — الوقت، المكان، المقاعد، فتح الخريطة، مشاركة
3. **الانضمام** — الاسم + جوال اختياري، يُحفظ محلياً على الجهاز

## التقنية

- Expo SDK 57 + Expo Router
- IBM Plex Sans Arabic
- Reanimated للحركة
- AsyncStorage لحفظ الانضمام (بدون خادم في هذه المرحلة)

البيانات الحالية تجريبية (mock) لأحياء الرياض.
