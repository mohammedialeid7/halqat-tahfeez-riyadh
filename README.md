# حلقات تحفيظ الرياض

تطبيق React Native (Expo) بسيط لاكتشاف حلقات تحفيظ القرآن للرجال في الرياض والانضمام إليها — بالعربية واتجاه RTL فقط، بدون تسجيل حساب. حلقات الرجال فقط.

## التشغيل

```bash
npm install
npm start
```

ثم افتح على جهازك عبر Expo Go، أو:

```bash
npm run web
npm run android
npm run ios
```

## الشاشات

1. **الرئيسية** — قائمة الحلقات مع فلاتر الحي / الفئة العمرية / الوقت
2. **أضف حلقة** — إضافة حلقة رجال بدون حساب (تُحفظ على الجهاز)
3. **الدعم** — تواصل وبلاغات
4. **تفاصيل الحلقة** — الوقت، المكان، المقاعد، فتح الخريطة، مشاركة
5. **الانضمام** — الاسم + جوال اختياري

التنقل السفلي: الرئيسية · أضف حلقة · الدعم


## التقنية

- Expo SDK 57 + Expo Router
- IBM Plex Sans Arabic
- Reanimated للحركة
- AsyncStorage لحفظ الانضمام (بدون خادم في هذه المرحلة)

البيانات الحالية تجريبية (mock) لأحياء الرياض.
